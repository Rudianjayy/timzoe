Panduan Setting WIFI.UM dan WIFI_1X.UM Menggunakan Android

  • Diposting oleh Siska Novianti
  • Kategori Artikel, Wifi dan Jaringan
  • Tanggal 13 Maret 2018

SETTING WIFI_1X.UM

1. Masuk ke setting

2. Pilih Wiresless Network

3. Akan muncul beberapa pilihan wifi. Pilih WIFI_1X.UM

4. Saat memilih WIFI_1X.UM pada smartphone anda akan muncul pilihan-pilihan security seperti yang nampak pada gambar. Berikut petunjuk pengisiannya:

  • EAP Method : PEAP
  • Phase-2 Authentication : None
  • CA Certificate : (unspecified)
  • Indentity : “Isi dengan NIP Anda”
  • Anonymous Identity : Kosongi
  • Password : “Isi dengan password akun tunggal (akun tunggal biasanya digunakan login saat pengisian kinerja.um.ac.id (ppkpns) atau siakad.um.ac.id)”
  • Klik Connect

5. Setelah sukses, Anda sudah bisa menggunakan Internet.

 

SETTING WIFI.UM

1. Masuk ke setting

2. Pilih Wireless Network

3. Akan muncul beberapa pilihan wifi. Pilih WIFI.UM

4. Buka browser, kemudian coba untuk browsing, misalnya ke google.com. Maka anda akan diarahkan ke halaman autentifikasi. Pilih Advanced. Kemudian pilih “Proceeed to 192.0.2.1 (unsafe).

5. Kemudian akan muncul halaman Login.

Masukkan Username dan Password. Username dan Password yang dimaksud disini adalah username dan password sesuai dengan akun tunggal, biasanya digunakan untuk login ketika akan mengisi kinerja.um.ac.id (PPKPNS) atau siakad.um.ac.id. Klik Submit

6. Setelah sukses, anda akan diarahkan ke laman um.ac.id yang artinya anda sudah bisa menggunakan Internet di lingkungan Universitas Negeri Malang.

 

NOTE:

  1. WIFI_1X.UM dan WIFI.UM bisa digunakan oleh semua mahasiswa, dosen, maupun tenaga kependidikan Universitas Negeri Malang.
  2. Lebih baik menggunakan WIFI_1X.UM dibanding dengan WIFI.UM karena dimanapun ada WIFI_1X.UM smartphone anda otomatis tersambung dengan WIFI tersebut tanpa melakukan penyettingan ulang (seperti memasukkan username dan password).
  3. Jika sudah bisa tersambung dengan WIFI_1X.UM atau WIFI.UM namun masih belum tersambung dengan Internet silahkan masuk ke setting -> wireless -> WIFI_1X.UM atau WIFI.UM -> Forget Network. Kemudian silahkan setting mulai awal.
  4. Jika ada peringatan username atau password salah silahkan cek kembali NIP dan password akun tunggal anda. Jika benar-benar yakin bahwa username dan password anda benar namun masih ada notifikasi bahwa username atau password salah silahkan ke Helpdesk TIK. Gedung G1 Lantai 1.
  5. Dilarang menginformasikan atau memberikan akun tunggal (username dan password) kepada orang lain. Keamanan akun tanggung jawab masing-masing pemilik akun. Pihak TIK tidak bertanggung atas penyalahgunaan akun tersebut.
